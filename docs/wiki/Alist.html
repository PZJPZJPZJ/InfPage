官方地址

  • 官网
  • GitHub

DockerCompose部署

services:
    alist:
        image: xhofe/alist:latest
        container_name: alist
        volumes:
            - ./config:/opt/alist/data
        network_mode: host
        restart: always

运行命令

# Docker
docker exec -it alist ./alist admin set NEW_PASSWORD
# Windows
.\alist.exe admin set NEW_PASSWORD
# Linux macOS
./alist admin set NEW_PASSWORD
